# Gini Index Prediction

This repository contains a Jupyter notebook for predicting the Gini Index
using economic data.

## Files

- Gini Index Prediction.ipynb — main notebook
- economicdata.csv — dataset

## How to use

### Online via Binder

Launch the notebook in your browser without installing anything:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/swarit930/GiniIndex/main?filepath=Gini%20Index%20Prediction.ipynb)

### Locally

1. Install Python 3.10+
2. Install required packages:

```bash
python3 -m pip install -r requirements.txt
```

3. Open the notebook:

```bash
jupyter notebook "Gini Index Prediction.ipynb"
```

## Notes

- Ensure economicdata.csv is in the same folder as the notebook.
- This repo is intended for educational purposes.
